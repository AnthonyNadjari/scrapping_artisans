================================================================================
TEMPLATES WHATSAPP - MESSAGES POUR ARTISANS
================================================================================

Ce fichier contient tous les templates de messages WhatsApp utilisés pour
contacter les artisans. Les variables {metier}, {ville}, {note}, {nombre_avis}
seront remplacées automatiquement lors de la génération des messages.

================================================================================
1. REPUTATION × Pas de site
================================================================================

Priorité: 100
Condition: Pas de site web + Note >= 4.0 + Nombre d'avis >= 5

Bonjour,

Je suis tombé sur votre fiche Google en cherchant un {metier} sur {ville}. J'ai vu que vous avez {note}/5 avec {nombre_avis} avis, mais malgré ça, vous n'êtes pas apparu en haut des résultats !

NOTES IMPORTANTES
================================================================================

- Les variables {metier}, {ville}, {note}, {nombre_avis} sont remplacées automatiquement
- Si note ou nombre_avis sont absents, les références sont supprimées du message
- Les sauts de ligne multiples sont automatiquement réduits
- Le prénom peut être ajouté automatiquement si détecté dans le nom de l'entreprise
- Les liens markdown sont conservés pour WhatsApp
